Tag h1

Voltar

Em uma página web, sempre encontramos frases em que se parecem com um título, e para isso usamos a tag "h1" (envolvidas entre < >). Pois assim, temos a frase evidenciada, sinalizando um título

É sempre importante usá-la quando precisamos destacar algum tipo de notícia, por exemplo, sendo o h1 usado para mostrar um Cabeçalho

Tag h2

Voltar

Já o uso da tag h2, serve para mostrar sub-títulos.

Tag p

Voltar

A tag p, serve para quando precisamos mostrar um parágrafo.

Tag ul e ol

Voltar

Ambas as tags ul e ol (envolvidas sempre entre <>) servem para mostrar uma lista. E você deve estar sem perguntando: qual seria a diferença entre elas? A tag ul mostra uma lista desordenada e já a tag ol exibe uma lista ordenada

Tag href

Voltar

A tag href serve para ancorar ou linkar uma página em seu próprio site ou mesmo abrindo um link externo.
